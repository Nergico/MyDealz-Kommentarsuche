# MyDealz-Kommentar-Volltextsuche-
Dieses Tampermonkey-Skript rüstet eine praktische Volltextsuche für Kommentare auf MyDealz nach. Bei Deals mit hunderten Antworten hilft dir die neue Suchleiste direkt über den Kommentaren, bestimmte Begriffe oder Erfahrungen blitzschnell zu finden. Treffer werden farblich hervorgehoben. Spare dir endloses Scrollen!
